## Section 5: Process

**Overline**: "How We Work"

**Headline**:
"From audit to operating model in 90 days"

**Subheadline**:
A phased approach that delivers compounding value. No big-bang launches. No surprises.

### Steps

**1. Discovery**
Audit systems, map processes, interview stakeholders. Understand what's working, what's broken, and what's missing entirely.

**2. Design**
Architecture decisions, data model, workflow design. We document the ideal state before touching a single setting.

**3. Build**
Hands-on implementation. CRM configuration, automation setup, dashboards, integrations. We build it in your systems, not in a sandbox.

**4. Launch**
Team training, documentation, go-live support. Nobody gets left behind.

**5. Optimize**
Ongoing iteration, reporting reviews, system refinement. The work doesn't stop at launch.

### Design Direction
- Numbered step blocks (1–5) with title + description
- Vertical timeline or horizontal stepper layout
- Subtle connecting lines between steps
- Reveal animation on scroll
